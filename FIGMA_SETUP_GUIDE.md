## Figma Tips

### Organizing Your Design System
1. Use consistent naming conventions
2. Create component variants
3. Use auto-layout for responsive components
4. Set up proper constraints
5. Use styles for colors and typography

### Export Settings
- Icons: SVG format
- Images: PNG/WebP format
- Use 2x resolution for retina displays
- Optimize file sizes

### Collaboration
- Share with team members
- Use comments for feedback
- Version control with Figma's history
- Export assets for development
